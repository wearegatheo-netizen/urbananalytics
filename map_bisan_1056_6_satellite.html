🔑브이월드 인증▾

ID/PW/API 키는 이 PC 브라우저에만 저장됩니다(localStorage). 지적도 표시·지번 선택·연속지적 실행에 VWorld API 키가 필요합니다. 법제처 OC 키를 넣으면 조례 건폐율을 자동 조회하며, 없으면 용도지역별 시행령 상한을 적용합니다.

🗺️연속지적 및 용도지역▾

주소 반경 내 연속지적(필지 경계) + 용도지역을 처리합니다. ‘DXF 다운로드’는 웹에서 바로 CAD용 DXF(EPSG:5186, 필지+용도지역)로 받고(QGIS 불필요), ‘연속지적 실행’은 로컬 스크립트로 저장 폴더에 전체 산출물을 생성합니다(브이월드 로그인 필요). 둘 다 VWorld API 키가 필요합니다.

        🏢블록별 평균높이 산정▾
        

          
필지를 고르는 방법을 선택하세요. 건폐율·도로폭·필지깊이는 자동 산정되며, 아래 설정으로 덮어쓸 수 있습니다.


          
          

            
            
          


          
          

            

              
              
지도 선택 필지 0개

              

              

                
                
                
                
              

              
여러 필지를 체크한 뒤 ‘획지로 묶기’를 누르면, 합쳐진 경계 기준(면적 합·접도길이·깊이)으로 평균높이를 산정합니다.

            

            

              
              

              
전(전면)·후(후면) 후퇴는 비우면 전역 기본값을 사용합니다. ‘용도지역 상향’을 지정하면 그 획지의 건폐율·최고높이 상한을 상향 용도지역 기준으로 산정합니다.

            

          


          
          

            

              
              
            

            

              

                
                
              

              

                
                
              

            

            
리스트에 좌표·주소가 있으면 접도길이·필지깊이는 자동 산정됩니다. 공통값으로 덮어쓸 때만 입력하세요.

          


          
          

            건폐율 · 건축선후퇴 · 사선계수 자동값 덮어쓰기(선택)▾
            

              
건폐율은 용도지역 → 지구단위계획·조례(법제처)·시행령 상한 순으로 자동 적용됩니다. 확인값으로 덮어쓸 때만 입력하세요.

              

                

                  
                  
                

                

                  
                  
                

              

              

                

                  
                  
                

                

                  
                  
                

              

              
전면/후면 후퇴는 전역 기본값입니다. 획지별로 다르게 적용하려면 지도 선택 목록의 ‘획지별 건축선후퇴’를 사용하세요.
